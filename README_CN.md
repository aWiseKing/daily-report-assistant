# DAILY REPORT ASSISTANT (日报助手)
[English](README.md) | [中文](README_CN.md)

## 🌟 项目介绍

**日报助手 (Daily Report Assistant)** 是一个旨在简化日报管理流程的高效 Skill。它可以帮助你自动创建、管理和组织 Markdown 格式的每日工作日志。

无论你是需要记录代码提交的开发人员 👨‍💻，还是需要更新项目进度的产品经理 👩‍💼，或者是任何需要记录每日成果的职场人士，这个 Skill 都是你的得力助手！🚀

## 🎯 适用场景

- **自动生成日报**：一句话指令即可创建当天的日报文件。
- **轻松更新内容**：随时随地添加今日完成的工作、遇到的问题或明天的计划。
- **智能管理条目**：通过简单的自然语言指令，修改或删除特定的日报条目。
- **结构化记录**：自动生成带编号的列表和分类板块，保持日报格式整洁统一。

## 🛠️ 配置与安装

以 **Kimi** 智能助手为例，安装步骤如下：

1.  **定位 Skills 目录**：找到你的 Skills 存放目录，通常位于 `~/.kimi/skills`。
2.  **复制项目**：将 `daily-report-assistant` 文件夹复制到该目录下。
    ```bash
    cp -r daily-report-assistant ~/.kimi/skills/
    ```
3.  **验证文件**：确保文件夹中包含 `SKILL.md` 文件，它定义了 Skill 的核心能力。

### ⚙️ 个性化配置

你可以通过修改 `SKILL.md` 或在对话中直接指定来调整默认设置：

- **默认存储路径**：`G:\file\工作汇报\周报` (可根据你的实际环境修改)
- **文件命名格式**：`日报_YYYY_MM_DD.md`

## 📖 使用指南

安装完成后，直接与助手对话即可管理你的日报！🗣️

### 1. 创建日报
> “构建今天的日报”
> “生成 2023-10-27 的日报”

这将生成一个包含标准板块的文件 `日报_2023_10_27.md`：
- `## 今日完成`
- `## 问题与风险`
- `## 明日计划`

### 2. 添加条目
> “添加：完成了登录页重构”
> “记一笔：修复了支付回调的 bug，放在问题板块”

助手会自动追加带编号的条目：
`- [#1] 完成了登录页重构`

### 3. 修改条目
> “把 #1 改成：完成了登录页重构并补充了单元测试”

### 4. 删除条目
> “删除 #2”
> “把关于支付 bug 的那条删掉”
